# Freelancer Landing Web
## Descripción
Este es un proyecto de una landing web tipo plataforma freelance.
## Características
Las tecnologías usadas son:
- HTML
- CSS
Diseño responsive:
- Navegación interactiva
- Barra de menú funcional
- Secciones visuales claras
Cómo ver el proyecto
1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. ¡Listo!
## Demo 
Puedes ver la demo del proyecto aquí:
